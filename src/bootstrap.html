Bootstrap

Bootstrap är ett populärt front-end ramverk som erbjuder möjligheten att skapa enkla och responsiva webbplatser. Ramverket tillandahåller bland annat färdiga knappar, formulär och JavaScript-komponenter. Det finns också inbyggda stöd för att webbplatser ska kunna fungera fint på alla enheter.

Jag tyckte om ramverket. Jag har knappt någon erfarenhet kring ramverk så jag kan inte jämföra direkt men jag tyckte att det var väldigt logiskt. De färdiga formulären var väldigt stilrena och bra. Dessa kunde också anpassas rätt enkelt med hjälp av deras olika klasser. Jag kan dock se att detta är ett ramverk där man inte kan bli allt för kreativ då dess design ses på de allra flesta webbplatser.
