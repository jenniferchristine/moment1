Publicering

Jag har valt att publicera min webbplats via Netlify. Netlify stödjer bland annat HTML, CSS och JavaScript och är optimerad för ramverk som till exempel React, Angular, Vue.js och Gatsby. Deras tjänst finns i olika omfattningar och kostnader. Gratisplanen som jag använder erbjuder oändingt med antal statiska sidor men också bland annat deployment från Github och 100GB bandbredd per månad. En betalplan kostar från omkring 200 svenska kronor och innefattar då mer av det som erbjuds i gratisplanen samt bättre support och snabbare åtkomst.

Jag tycker att Netlify är väldigt enkelt och smidigt att använda. Detta mycket för att man kan uppdatera en befintlig webbplats direkt vid snabba ändringar genom att pusha dessa via Git. Netlify erbjuder också ett globalt CDN som gör att alla webbplatser är lätta att komma åt, oavsett var man befinner sig. Jag tycker också att deras gratisplan är väldigt bra då den räcker långt för många. Om man däremot skapar en stor applikation så kan Netlify vara ett sämre val då dom är just optimerad för statiska sidor och detta kan därmed bli dyrt i längden om man behöver uppgradera för att det ska täcka serverbehovet.

Man kan enkelt publicera via Netlify genom att endast ladda upp sina filer. Jag föredrar dock att "deploya" via Git. Detta görs genom att skapa ett repo på till exempel Github. Hit pushar man sin kod för att sedan länka sitt konto till Netlify. Därefter kan man sedan se sina publika repon och välja ett. Netlify bygger och publicerar webbplatsen per automatik när något valts eller vid commit.
